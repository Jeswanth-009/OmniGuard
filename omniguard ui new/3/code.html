GET /api/v2/data/streams

Real-time Data Stream

Initialize a multiplexed binary stream from the OmniGuard gateway. This endpoint supports WebSocket upgrades and gRPC-Web for low-latency telemetry.

Parameters

Key Type Description
stream_id UUID Unique identifier for the session.
buffer_size Integer Size of the initial flush window (kb).
REQUEST SAMPLE 
curl -X GET \
  'https://api.omniguard.io/v2/data/streams' \
  -H 'Authorization: Bearer YOUR_TOKEN' \
  -d '{
    "stream_id": "8f3e-92b1-4f22",
    "buffer_size": 1024
  }'
RESPONSE 200 OK 
{
  "status": "established",
  "connection": {
    "protocol": "wss",
    "node": "us-east-4",
    "latency": "8ms"
  },
  "metadata": {
    "shards": 12,
    "security": "AES-256-GCM"
  }
}
